# Income Expense Calculator

## Description
A simple application to track and manage income and expenses, helping users calculate their financial balance.

## Features
- Add income entries
- Add expense entries
- Calculate total income
- Calculate total expenses
- View net balance (income - expenses)
- Clear history

## Installation
1. Clone or download the repository
2. Navigate to the project directory
3. Install dependencies (if any)
4. Run the application

## Usage
1. Start the application
2. Enter income amounts with descriptions
3. Enter expense amounts with descriptions
4. View the calculated totals and balance
5. Export or save your records

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
```
├── index.html
```

## How to Use
- Input field: Enter the amount
- Category: Select income or expense
- Description: Add a note
- Submit: Click to save entry
- View: All transactions displayed in list
